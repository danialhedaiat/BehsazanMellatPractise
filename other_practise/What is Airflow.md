# What is Airflow (Apache Airflow)
## intrduction
Apache Airflow is a service for orchestrating and scheduling tasks or jobs. 

## pros
- it can start scheduled task 
- it can run background worker to start for some sequence of jobs
- python base
- monitor and track task

## cons
- use for batch processing
- complex setup
- python base
